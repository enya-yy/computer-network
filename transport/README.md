## UDP: User Datagram Protocolf (rfc768) <a name = "udp"></a>

基于IP协议做了多路复用/分用，简单错误检测。

UDP属于“Best effort”服务，他尽可能的传输数据。从而可能会产生丢失和数据的非按需到达。
